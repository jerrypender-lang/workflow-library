# HFA / AI Aunties Workflow Library

**A curated library of reusable business automation workflows for small businesses, nonprofits, and creators.**

## Overview

This workflow library is a collaboration between Human First Automations (HFA) and AI Aunties. It contains ready-to-use workflow templates covering operations, marketing, client management, and content creation.

## What's Included

- Business operations workflows
- Marketing and outreach automation templates
- Client onboarding and communication flows
- Content creation and scheduling systems
